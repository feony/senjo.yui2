# Senjo Yuki Conveyor

Это платформа (фреймворк) эмулирующая исполнение большого числа виртуальных thread потоков. Главные критерии качества — скорость выполнения и экономия ресурсов. В конвейере такой независимо выполняющийся алгоритм называется Task. Он является очень компактным примитивом, его работа реализуется разработчиком, состоит из отдельных этапов, порядок выполнения которых может определяться и по ходу работы. Задача может получать и обрабатывать сигналы извне, может работать по таймерам или бесконечно зацикленно. Может завершить работу когда посчитает нужным, либо эмулировать работу службы и выполняться бесконечно работая над заданием или в фоне ожидая команд.

Более подробную документацию разрабатываю и веду здесь: http://task.senjo.org/yuki
